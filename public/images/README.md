# Logo Instructions

Please add your logo file here as `logo.png`.

The logo will be displayed in the header navigation bar.

**Requirements:**
- File name: `logo.png`
- Format: PNG (with transparent background recommended)
- Recommended size: Height around 52px (width will scale automatically)
- Location: `/public/images/logo.png`

The logo path is already configured in `components/Header.js` to use `/images/logo.png`.
